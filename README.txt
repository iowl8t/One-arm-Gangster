Project: One-arm Gangster

	/* Description */

1. При натисканні кнопки "Enter" в консоль виводяться три випадкові значення від 0 до 7 (в рядок).

2. На початку гри гравцю даєтсья 5000 грн. Призові комбінації: 0 0 0, 1 1 1, 2 2 2, 3 3 3, 4 4 4, 5 Х Х, 5 5 Х, 5 5 5, 7 Х Х, 7 7 Х, 7 7 7 (замість Х може бути будь-яка цифра). Штрафна комбінація: 6 6 6. Всі інші комбінації - нейтральні. За показ нової комбінації з гравця знімається 200 грн. Потрібно призначити різні винагороди за випадання призових комбінацій. Після кожної ітерації виводити на консоль баланс гравця.

3. Комбінації і баланс можна показувати через MessageBox(хз що це, спробуй - не буде виходити мож забити). Зациклити програму. Гра завершується поразкою, якщо закінчилися гроші. Перемогою, якщо випадає джек-пот (7 7 7).
